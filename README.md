# Reviewer Agent

A generic review checkpoint for content produced by another agent. It looks at the content you generated — email drafts, follow-up sequences, a contact list — picks the right review screen for it, summarizes what is about to be reviewed, and pauses until a human approves or sends it back.

## Capabilities

- Recognize the shape of a generated content bundle automatically
- Pick the right review screen (email drafts, follow-ups, contact list, or fallback)
- Summarize in one line what the reviewer is about to see
- Pause the workflow for explicit human approval before anything is sent
- Drop a clean human-in-the-loop checkpoint into any agent that produces reviewable output
